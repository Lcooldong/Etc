git commit -m "원하는 말"

3. 한이음 gitlab 홈페이지 확인




4. 브랜치

git branch {원하는 브랜치 이름}

git branch  <- 브랜치 확인

git checkout {원하는 브랜치 이름}   <- 해당 브랜치로 전환

git checkout -d {원하는 브랜치 이름}  <- 브랜치 삭제


<브랜치 올리기> (-u 새로 추가 기능)

git push -u origin {원하는 브랜치 이름}  <- 처음 만들기 브랜치 홈페이지에 올리기
git push origin {원하는 브랜치 이름} <- 이후 부터는 그냥올릴 수 있음(삭제후 만들때도됨)


이후 작업은 
옮긴 브랜치에서 위 2번 add, commit push 수행하면됨





5. 브랜치 병합 요청  (본인 브랜치 -> master)

홈페이지 좌측 메뉴

Merge Request 누르고 New merge request

본인 브랜치 - > 타겟 브랜치


설명 쓰고 다른 거 딱히 안눌러도됨





6. 병합 요청 수락하기 

Merge Request 누르고 해당 요청 누르고 Merge 버튼 누르면 해결
